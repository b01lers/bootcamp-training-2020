Basic Devtools Example

Press F12 or right click -> inspect element to open the built in dev tools for your browser!

Inspect Element

Inspect element is one of the most useful tools for web challenges

It's got a bunch of useful tools

  1. Inspector: View all of the html and inline css and javascript
  2. Console: To run javascript in your local browser
  3. Debugger: A built in javascript debugger
  4. Style Editor: Views css
  5. Performance: Shows cpu usage
  6. Memory: Shows memory usage
  7. Network: View the full request and response for everything the page loads
  8. Storage: Shows cookies that the site is storing on your computer
  9. Accessibility: Info for screen readers
