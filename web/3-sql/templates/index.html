SQL is an important tool for websites. It is a fast and easy way to display specific and dynamic information to the end user.

In this exmample we will be going over relational databases only (specifically MySQL). A relational database is a database that consists of databases, tables, pre defined columns and rows.

  • Databases

    Multiple databases can exist in the same instance. These databases store different tables

  • Tables

    Tables store information in columns and rows

  • Columns

    Columns have a set layout with a name, type (string, int), and can have a max or min size. You can think about this as pre defining a variable you will use later.

  • Rows

    Rows are entries in the database table, this contains all of the information in the database organized how the columns are described
