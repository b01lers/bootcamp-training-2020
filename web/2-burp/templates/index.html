Repeater

The repeater is a very useful part of burp, it allows you to change the request until you get something that works

The easiest way to set it up from an intercepted request. Type in any username and password, then click 'Login' to send a post request.

Once you see it in the Proxy --> Intercept window, click 'Action', then 'Send to Repeater'

Click the repeater tab and you should see the request on the left and an empty response on the right.

Use the 'send' button to send the request and update the response box
